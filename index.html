인공위성 천체 관측 방해 예측기

CelesTrak의 공개 OMM 궤도 데이터와 SGP4 계산을 이용해 관측 천체 주변을 통과하는 위성을 찾고, 촬영 시각을 조정했을 때 위험 촬영이 얼마나 감소하는지 분석합니다.

위성 데이터 없음 먼저 최신 자료를 불러오세요.
1

위성 데이터 준비

최신 Starlink OMM 자료를 불러오거나 내려받은 JSON 파일을 선택합니다.

CelesTrak 서버 또는 브라우저의 교차 출처 정책 때문에 자동 불러오기가 실패하면, CelesTrak에서 OMM JSON을 내려받은 뒤 두 번째 버튼으로 파일을 선택하면 됩니다.

2

관측 조건 설정

날짜는 궤도 데이터의 기준 시각과 가까운 날짜를 사용하는 것이 안전합니다.

분석을 준비하고 있습니다.

해석 범위와 한계

  • 위성의 실제 겉보기등급과 카메라 센서 포화 정도는 계산하지 않습니다.
  • 위성이 지구의 원통형 본그림자 안에 있는지를 근사하여 태양 조명 여부를 판정합니다.
  • 구름, 미세먼지, 광공해, 건물과 산은 반영하지 않습니다.
  • 선택한 위성 표본과 시간 범위에 대한 탐색적 분석이며 전체 위성군의 보편적 결론은 아닙니다.
